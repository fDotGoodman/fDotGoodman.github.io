Sensitivity Analysis of an Agent-Based Artificial Immune System

This work proposes a novel artificial immune system as an agent-based simulation, implemented through Repast Simphony. It evaluates the sensitivity of the model to its initial parameters, encapsulating network intrusion data as pathogens in the system.

Evaluating the Effectiveness of Different Machine Learning Classifiers at Detecting Forest Fires in a Near Real-Time Environment

This paper looks into different machine learning approaches to determine the best starting point for detecting forest fires in near real-time. A Decision Tree, a Support Vector Machine, Logistic Regression, and a Gradient Boosted Tree are analysed on the Fires from Space: Australia dataset.

Projects

Here are some example projects I've undertaken, both as part of my studies, and in my free time. Though not all of these are publicly available, let me know if you'd like to see the codebase or a demonstration.

B-Lymphocyte Simulation

This Java-based simulation, implemented through Repast Simphony simulated the process of biological somatic hypermutation, with a view to apply it to network intrusion detection.

Comparison of Classifiers on Forest Fire Detection through Apache Spark

This work compared classification baselines of a range of supervised classification models on fire prediction on an Australia-based wildfire dataset.

Rubbish Collection Simulation through Genetic Algorithm

This agent-based simulation, implemented through Repast Simphony, looked to incorperate a genetic algorithm to intelligent agent rubbish collection in a spatial coordinate system.

MERN Stack Digital Display Projection System

This full-stack web-development project implemented a digital display system projection system to schedule the broadcast of a range of media onto a surface.
