B-Lymphocyte Simulation

This project accompanies the paper titled Sensitivity Analysis of an Agent-Based Artificial Immune System. It involved implementing an novel artificial immune system through the Repast Simphony framework for Java. This AIS was written with a view to classify network intrusions, seeing possible use in a conceptual distributed intrusion detection system. The system itself acts as a model of the adaptive mammalian immune system. The corresponding innate immune system was implemented as a Support Vector Machine classifier written in MATLAB to classify the intrusions, then exporting them for filtering and memory modelling in the simulation. The model was found to be sensitive to all parameters tested.

This project was completed as part of my undergraduate degree, and rights are therefore held by The University of Nottingham.
