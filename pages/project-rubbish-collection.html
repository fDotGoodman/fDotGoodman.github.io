Rubbish Collection through Genetic Algorithm

This project was written in Java using the Repast Simphony agent-based modelling toolkit. It looks to see how reactive agents behave when tasked route identification and optimisation of the renown Travelling Salesman Problem. The agents involved include Collectors who map out the spatial coordinate system, building up their own internal memory of where rubbish lies. They then perform a memetic algorithm to determine the most optimal route through the coordinate system to collect them, and finally action this route. Various runtime parameters are ported to the Repast GUI, allowing the user to interact with the simulation, changing them, observing agent behaviour through a view of the spatial system, and witnessing data trends through statistical graphs generated in runtime.

This project was completed as part of my undergraduate degree, and rights are therefore held by The University of Nottingham.
